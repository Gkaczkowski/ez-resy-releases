Installation

macOS

  1. Download the .dmg file for your Mac (Apple Silicon or Intel)
  2. Open the .dmg and drag ez-resy to Applications
  3. Right-click the app and select "Open" (not double-click)
  4. Click "Open" in the dialog that appears to bypass Gatekeeper

Why is this needed? This app isn't code-signed with an Apple Developer certificate. macOS Gatekeeper quarantines unsigned apps downloaded from the internet.

Still seeing "app is damaged" error? Open Terminal and run:
xattr -c /Applications/ez-resy.app Then right-click and Open the app again.

Windows

  1. Download the .exe installer
  2. Run the installer and follow the prompts
  3. If SmartScreen warns you, click "More info" then "Run anyway"

Linux

  1. Download the AppImage or .deb file
  2. For AppImage: chmod +x ez-resy-*.AppImage && ./ez-resy-*.AppImage
  3. For .deb: sudo dpkg -i ez-resy_*.deb

Setup

  1. Open the app and go to Settings
  2. Click Connect to Resy and log in with your Resy account
  3. Add restaurants you want to monitor from the Search page
  4. Configure your preferred time windows and party size
  5. Enable the scheduler and let ez-resy do the work!
